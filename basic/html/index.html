폼 (Form)

서버에 제출하는 데이터를 포함한다

Google

    1 form 
    클라이언트의 데이터를 서버에 전송한다
    예) 로그인 폼, 가입폼, 검색폼, 컨택폼 등

    - 애트리뷰트
    1) action: 폼을 제출할 서버상의 주소

    2 input
    실제로 유저의 데이터를 입력받는다

    - 애트리뷰트
    1) type 
    수집하고자 하는 데이터에 따라 다양한 타입이 존재한다

    2) name
    서버가 폼데이터를 분류하는 기준

    3) placeholder
    입력해야하는 데이터의 예시를 보여준다

    3 button
    폼을 제출하는 버튼

    - 애트리뷰트
    1) type
    버튼의 목적
